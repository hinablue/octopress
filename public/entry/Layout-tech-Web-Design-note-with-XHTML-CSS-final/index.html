它還是 Click Me 也沒什麼稀奇的，只是字越來越大而以。那麼，不知道各位會怎麼製作這樣的按鈕？當然，用圖片是最快的做法，不過，這四個按鈕你需要用多少張圖片呢？12 張？免！8 張？免！6 張？免！ericsk 告訴我們，一張都不用！為什麼？
