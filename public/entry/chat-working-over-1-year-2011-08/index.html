[閒聊] 週年慶一下

Aug 1st, 2011

到職好像恰好一年了。

如果你要問我有沒有什麼改變？大概是這樣吧：

  • Javascript 比 PHP 還熟（我已經不會寫 PHP 了
  • Ubuntu 還算上手了（BSD 哭哭
  • 用了 CakePHP，寫過 CodeIgniter，對 Zend 還是不太熟
  • ORM 好雷（因為 CakePHP 本身就是採用 ORM
  • 接觸了 Android 的開發
  • 累積了超大量文件，然後都看不完（噴淚
  • 買了很多書
  • HTML5, CSS3, SASS, NodeJS, Python

因為老闆是外國人，所以英文好重要（但是我英文依舊很爛），不過因為老闆會說中文所以還可以溝通（這是自暴自棄吧）。如果硬是要說有沒有什麼感想或是想法，那大抵上應該是這樣：

  • 需要 Crazy idea
  • 要學得東西好多
  • 時間只有這樣（這是擠不出來的
  • 認識好多神人
  • 覺得自己好渺小
  • 如果可以，我需要人生重來槍
  • 時間、時間、時間、時間、時間
  • 唸書、唸書、唸書、唸書、唸書

工作是這樣，我們用既有的，不對，應該說是用自身的經驗累積，來去換取一份能夠學習，或是更進一步往前走的環境，或者是，講白話一點，講市儈一點，就是換取一份能夠養家餬口的薪水。我不否認這是現實。只是，我們都是在這樣的環境繼續往前，或者說推著自己往前。我也不否認實際上可能的怠慢、倦惰或是逃避等等的情緒。

以前有在看我的 Plurk 的人大概知道我脾氣不是很好（無誤），情緒是這樣，遇到那些事情，我要是沒有一個管道抒發的話，我應該真的會瘋掉。你說我 EQ 不好也無所謂，我只不過是把負面的東西找個地方傾倒而已。只不過可能傷了大家的眼就是，真是對不住啊（一拜）。

回想起來，從一開始工作就是救火隊的經歷，真是多采多姿啊（眼神死）。直到這個工作，才讓我恢復正常的工程師的位子（倒）。你說不好嗎？其實也沒有不好！畢竟以前那些風風雨雨，還真的讓我學到了很多，除了工作以外的事情，一些關於人與人之間的事情。例如凌晨三點接到機房打電話來，陳先生，你們的伺服器掛點了這樣。

不過也快 32 歲了（掩面）。

以前都覺得 32 是個很奇妙的數字，總覺得應該要做一些什麼大作為的事情。不過實際上似乎不是如此耶！我好像還沉浸在永遠看不完的新技術，永遠看不完的新書，永遠看不完的 change log 裡面。想了想，似乎應該抓緊一些什麼，然後發了瘋似的去衝刺才對。只是，如果像以前一樣搶不到套件，Code 到三點，好像又有點太超過。

我覺得 @pct 就是這樣的神人！一個為愛 BSD 而瘋狂的神人！

相較之下我只是耍耍嘴皮子的 BSD 使用者而已（遮臉）。認識這些神人真的是有趣的事情，@jaceju，大公，高見龍，SDpower，皇上，@ericsk，丞相，還有公司的神人們，還有很多我沒有列出來的神人們。在個別專精的領域裡面，其實這些都是非常好的學習對象，這些都是讓人非常熱血的學習對象！

我也一直在想，是不是應該像以前唸書那樣，清醒而瘋狂，去追求一些讓人興奮的事情。也許是缺了點什麼，只是再找到那些事情之前，還是得讓自己保持那樣的姿態，繼續朝著既定的方向前進下去！也許就像是 3 idiots 所說，追求卓越，成功自然就會跟著你。也或許，只是像失敗者，渴望飛翔。

其實，只是個擁抱而已。

各位，就繼續為了自己而瘋狂吧！
