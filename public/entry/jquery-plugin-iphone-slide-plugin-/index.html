[jQuery Plugin.] iPhone-slide Plugin 爆肝更新

Mar 20th, 2011

gitHub 在這裡：https://github.com/hinablue/jqiphoneslide DEMO 在這裡：http://jquery.hinablue.me/jqiphoneslide/

我拔掉一些東西：

  1. 滑動到指定頁面
  2. 無限頁面滑動
  3. 動態載入頁面（Ajax）
因為我還沒有實際測試所以先不放。

重新檢視了換頁演算方式，發現可以寫的比之前簡單點，所以就動手改了！然後… 凌晨三點多 Ubuntu 瞬間當掉讓我感到絕望啊！ 早上起來重新寫了一些東西，然後就丟上 gitHub 了。

然後 iPhone4 上面是可以跑的，只是效能不怎麼樣就是，關於這個在 Android 裡面跑也是一樣慢。不過我沒有 iPad，所以之前某個歪果人寫信來問我 iPad 上面能不能跑的問題，我也只能說抱歉了。

然後，我一直在想遞迴的問題，我覺得還是有地方能夠寫的更簡潔一點，但，改天吧（倒）。
