[Layout] deck.js + Html5-boilerplate 投影片產生器

Sep 6th, 2011

這年頭講求效率，所以有個一步安裝也是很合理的。

[code bash] $ wget –no-check-certificate https://github.com/hinablue/html5-deckjs/raw/master/install.sh -O - | sh [/code]然後就裝完了。

裝完之後，執行資料夾中的 html5-deckjs.sh，開始一鍵安裝。 [code bash] $ ./html5-deckjs.sh To create a new project, enter a new directory name:

app123

[/code]然後也就裝完了。

接著到你的家目錄底下去找 app123 的目錄，就是我們建立好的基本樣板。你可以直接用瀏覽器，打開 app123/index.html 看看結果。不要跟我說沒圖沒真相，因為實在太簡單所以我不想附圖了（咬我啊。

基本上我佛心的附上了 style.sass 檔案，你可以自行運用修改。

歐，請按左上角的 fork me on gitHub，找一個叫做 html5-deckjs 的就是了。
