[Chtml] 中文版如何做愛(超標記語言)？

Jul 11th, 2008

從阿維那邊看到的東西，實在是太好笑了，雖然這麼說好像有點不太道德，但是看到那個元素列表實在讓我無法克制啊！超標記語言翻成英文叫做 HTML，我想這個大家應該不陌生(應該)，但是 CHTML 就沒聽過了吧！我平時有在 Google Code 閒晃的習慣，既然阿維說上面有，就去繞了繞。

與其說估狗大神萬能，倒不如說中國大陸萬萬網民萬能吧！ http://code.google.com/p/chtml/ 元素表在這邊，真的超噴飯的啦！ http://code.google.com/p/chtml/wiki/IndexOfElements 而且他也有屬於自己的 DTD 喔！ http://chtml.googlecode.com/svn/trunk/chtml.dtd 為了方便起見，所以我把轉換成繁體中文比較淺顯易懂一點點。

名字 中文名字 描述
A 啊 錨點
ABBR 縮寫 縮寫, 隱語 (如 “功夫網”)
B 粗 把字變粗
BIG 大 把字變大
BLOCKQUOTE 一坨語錄 一大坨語錄引用
BODY 身體 如何做愛文檔的身體
BR 斬斷 強制斬斷本行
BUTTON 按鈕 能被按下的按鈕
CAPTION 表格標題 表格的標題
CITE 出處 引用的出處
CODE 代碼 漢程式設計語言原始程式碼
COL 列 表列
COLGROUP 一小撮列 表列的集合
DEL 沒說過 刪除已經說過的話, 並且效果能和沒說過一模一樣
DIV 分塊 把東西分成一塊一塊的
EM 含淚勸說 用於對一小撮不明真相的群眾說明真相
FORM 表單 表單
H1 一哥專用標題 標題
H2 總理級標題 標題
H3 省部級標題 標題
H4 司廳級標題 標題
H5 縣處級標題 標題
H6 科長級標題 標題
HEAD 頭 如何做愛文檔的頭
HTML 如何做愛 文檔的根基
I 歪 把字打歪
IFRAME 打開天窗 在文檔中開個窗子貼小廣告
IMG 貼圖 貼了圖就不用說話
INPUT 發言請小心 不明真相的群眾進行一行發言用
GOU 勾 一個貌似無論你打不打勾伺服器端都會認為你打了勾的勾勾框. CHTML業界獨創!
INS 插 不管怎麼樣, 插
LABEL 標籤 用於給事情定性的文字
LEGEND 不許聯想 不能用於聯想的欄位
LI 條條框框 一條一條的
NOSCRIPT 不帶發言稿 不許發言稿運行的命令
OBJECT 對象 如何做愛文檔裡面的物件
OL 排名一定要分清楚先後的條條框框 就是有序的列表, 如領導就座順序.
OPTION 選項 假裝供選擇的
P 屁 段落
PRE 服從命令聽指揮 不許裡面的文本格式亂掉
SCRIPT 發言稿 CHTML 支持一定程度上的發言稿, 如 Java-發言稿, VB-發言稿
SELECT 請選擇 選舉
SMALL 小 讓文字渺小
SPAN 俯臥撐 在橋上做俯臥撐, 用於劃分小段內容.
STRONG 停頓鼓掌 一般用於領導講話, 請務必提醒領導不要錯誤地把標籤本身念出
STYLE 作風 用於規定機關作風和粉刷外牆
SUB 頭上標 上標
SUP 腳下標 下標
TABLE 表格 表格
TBODY 表身 表格的身體
TD 表身裡面的儲存格 表格的格子
TEXTAREA 請填入大段文字 供不明真相的群眾進行大段發言用
TFOOT 表腳 表格的腳
TH 表頭裡面的儲存格 表格頭部的格子
THEAD 表頭 表格的頭部
TITLE 標題 文檔的標題
TR 表行 表的行
UL 排名不分先後的條條框框 無先後的列表
ANTIGFW 反功夫網 傳說在這個標籤中的文字, 通過俯臥撐和打醬油召喚上古神獸, 安全穿越功夫網. 在中國互聯網上建議您盡可能多使用此標記. CHTML業界獨創!

我問了問我們家的設計，他說如果真的是這樣，他要改行去做平面設計了。這麼想想好像也是，也許許多設計的習慣不會像我們工程師一樣使用 source code 來去編輯，但是，如果真的應用這些東西的話，那麼應該會很恐怖。我這麼說好像可能會有人摸不著頭緒，舉個例子好了。

[肯定的] 物價永遠不會凋零，就像薪水永遠不會漲

Jul 8th, 2008

下班途中，躲開旁陀大雨的紛鬧，安安靜靜的在路口等著長達 99 秒的紅燈。人生大概就像這樣一直一直的向下倒數，一直到換了燈號，不是升天就是下地獄，只能右轉或直走而已，不要問我為什麼不能左轉，你以為死掉了以後只要去待轉可以再復活嗎？告訴你不可以啦！重點不是待轉可以復活(肯定的)，重點是我右後方的一對情侶？夫妻？兄妹？反正是一男一女的對話。

$女：你覺得那個 LV 怎樣？ $男：還好吧，我覺得另外 OOOXOXXOXX 比較好。(編按：我只有 LP 不知道什麼 LV 啦！) $女：真的嗎？可是另外一個比較便宜耶。 $男：四萬多。 $女：我覺得那個 OOOXOXXOXX 還好，還賣五萬多。(編按：林老師勒，我 LP 壓到爆也沒那麼多薪水啦！) $女：拜託，我很窮耶。 $男：所以…(編按：綠燈了。)

嗯，怎麼說呢。我只想說，那些挺著金老二含著金湯匙出生的人，不要再哭窮啦！現在這個社會到底發生了甚麼事情？怎麼明明就是生活不虞匱乏，每天都有人照顧的人在那裡靠北，難道這才叫做真理就是了？我真的不懂，老哥，我不懂啊！每天拼命到底是為了什麼呢？每個人總是都有一點想要去完成的夢想，總是會有一點必須要去堅持的初衷，總是會有一株永遠不會讓它熄滅的火苗。

[影像] 陽明山涓絲瀑布

Jul 7th, 2008

A16 + CPL 老實說，我以前用 CPL 的時候都不會遇到這種狀況，但是第一次在數位機身上使用 CPL 問題卻一堆。最主要還是光源散射的問題，這種感覺就像是 LCD 螢幕漏光一樣，某個角度會散射光源，造成整個畫面白茫茫一片。我以前用 EOS 3 的時候從來沒有遇過這種狀況，到底是跟鏡頭有關係？還是跟機身有關係呢？我在 Mobile01 也有看到相關的討論串，所以，我覺得機身或者是鏡片可能有絕大關係吧？但，這片 CPL 還是信乃達的說，真是無言。

菁山路的入山口，一路可以走到擎天崗，大概有 3.2 公里左右，涓絲瀑布大概在一半的位置。

地上藍色的，應該是自來水管線吧。

CPL 加工後的藍天，但其實，用 Picture Style 也可以辦到。
