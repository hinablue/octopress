[*NIX] 美妙的流量

Jun 17th, 2008

眼尖的人應該看得出來 WEB4 是在 M$ 系統底下。雖然這不是什麼精彩的流量圖，不過，只是用來抒發一下最近工作常常遇到的狀況。但是說常常好像有點太誇張，應該說每天吧(沒有誤)。最近這幾天主機又被一些莫名其妙的木馬入侵，當然，對象都是那些個 M$ 主機。WEB5 是我到了這個公司才新裝上的一台 BSD 主機，因為對外的機會少，自然比較不會遭殃。至於那幾台 M$ 嘛，我看 MIS 還在主控端周旋，不知道木馬清乾淨了沒(慘)？

我不是 MIS，我只是一個普普通通的網頁程式設計師。這樣講好像很機八無情，但是沒辦法，領多少薪水做多少事情，老子已經看開了，要協助當然是可以，但是要我負責就免談。當然我自己負責的主機出狀況的話，我當然會把它弄好，只是通常出狀況的都不是我那台主機，也許，M$ 的系統比較好攻擊吧(肯定的)。像是之前遇到的 javascript 木馬，MySQL 漏洞攻擊，PHP Shell Script 漏洞攻擊，幹，怎麼好像都在自爆我公司的漏洞似的，我的機器好像都沒遇過。

老實說，其實我 BSD 的防火牆也沒有設定的很周全(自爆)。該擋的擋掉而已，畢竟我不是專攻主機的人，其實只是嫩咖一個而已，湊合湊合著用啦。

歐，你問我上面那兩張圖是做什麼(眾毆)？那個是流量監測表(廢話)。Outbound，你知道的，就是那個意思(？)。這兩台機器使用的是一條 2M 對稱的 ADSL，那，看出笑點了嗎？嘖，這年頭要說笑話挺難的。2M ADSL 理想頻寬約略為 256KB，你看看 WEB4 的頻寬使用最大值，296.49KB(大驚)。

救火隊的碎碎念

Jun 12th, 2008

最近一大堆 TextCube 的文章我想應該看膩了吧，我自己都有點膩了。再者，算起來我好像有很長一段時間沒有寫什麼(編按：那是因為你自己發懶吧！)文章了，所以，午休時間要來發奮圖強嗎？也不是，手頭上有兩個燙手山竽，紐澳的案子，菲觀的案子，每個都趕著要上，每個都沒有定案，老實說上頭的──如果他們真的敢說自己是企劃的話──活動企劃搞得我很煩。不管怎麼問，一律都說客戶還沒決定，不知道不清楚不要問我，如果是在當兵我應該會挖很大的洞等他們來跳吧！

搞不好禮拜六日還要來加班，當然，想也知道我應該是不會做這種事情！印象中，其中還有一位一直想到我們部門做網路企劃，說實在的，這時候不請出學生會長大人不行了！ 想當網路企劃，你還差個十億萬年啊！所以這些事情真的是讓人非常不悅(肯定的)！

老實說底下都是碎碎念，要看不看就隨客官自由心證啦。

工作至今除了上次自己白目的賠錢了事事件之外，到現在也沒存──認真說起來區區 2 萬多而已──幾個錢，當然大抵上是拿回家的居多，畢竟家裡的某些開銷是遠超乎──光是糖尿病專用的營養補給品，每個月最少要 7K 上下──我的想像的，但是，這不是藉口就是了。誰叫自己白目──怪誰啊！怪你自己啊！──賠掉了存了兩年的積蓄呢。

至於我弟，住家裡，雖然我不知道他拿多少錢回家，其實我也不會問，反正他住家裡有個照應總是好的。其實我自己應該是屬於比較幸運的人了，比起 NZMA 還有助學貸款，我已經比他好很多了。卡費，永遠都在 10K 以下，頂多過年的時候幫家裡添購一堆東西而超過，但是也還在控制之內。
