[閒聊] 週年慶一下

Aug 1st, 2011

到職好像恰好一年了。

如果你要問我有沒有什麼改變？大概是這樣吧：

  • Javascript 比 PHP 還熟（我已經不會寫 PHP 了
  • Ubuntu 還算上手了（BSD 哭哭
  • 用了 CakePHP，寫過 CodeIgniter，對 Zend 還是不太熟
  • ORM 好雷（因為 CakePHP 本身就是採用 ORM
  • 接觸了 Android 的開發
  • 累積了超大量文件，然後都看不完（噴淚
  • 買了很多書
  • HTML5, CSS3, SASS, NodeJS, Python

因為老闆是外國人，所以英文好重要（但是我英文依舊很爛），不過因為老闆會說中文所以還可以溝通（這是自暴自棄吧）。如果硬是要說有沒有什麼感想或是想法，那大抵上應該是這樣：

  • 需要 Crazy idea
  • 要學得東西好多
  • 時間只有這樣（這是擠不出來的
  • 認識好多神人
  • 覺得自己好渺小
  • 如果可以，我需要人生重來槍
  • 時間、時間、時間、時間、時間
  • 唸書、唸書、唸書、唸書、唸書

工作是這樣，我們用既有的，不對，應該說是用自身的經驗累積，來去換取一份能夠學習，或是更進一步往前走的環境，或者是，講白話一點，講市儈一點，就是換取一份能夠養家餬口的薪水。我不否認這是現實。只是，我們都是在這樣的環境繼續往前，或者說推著自己往前。我也不否認實際上可能的怠慢、倦惰或是逃避等等的情緒。

以前有在看我的 Plurk 的人大概知道我脾氣不是很好（無誤），情緒是這樣，遇到那些事情，我要是沒有一個管道抒發的話，我應該真的會瘋掉。你說我 EQ 不好也無所謂，我只不過是把負面的東西找個地方傾倒而已。只不過可能傷了大家的眼就是，真是對不住啊（一拜）。

回想起來，從一開始工作就是救火隊的經歷，真是多采多姿啊（眼神死）。直到這個工作，才讓我恢復正常的工程師的位子（倒）。你說不好嗎？其實也沒有不好！畢竟以前那些風風雨雨，還真的讓我學到了很多，除了工作以外的事情，一些關於人與人之間的事情。例如凌晨三點接到機房打電話來，陳先生，你們的伺服器掛點了這樣。

Jul 26th, 2011

Compass

他的全名就叫做 Compass（揍飛）！是一個基於 SASS 上面的開發工具。功能就是幫你兜好一些既定的 CSS，當然，是以 SASS 的方式來作。既然是基於 SASS 的工具，也是一樣，請先準備好 Ruby 的環境，並且確認你已經把 SASS 給安裝好了。至於 Compass 的安裝方法，官方也有教。

$ gem install compass

compass create my_test_project --syntax sass
