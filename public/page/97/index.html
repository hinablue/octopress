[大神] Google 瀏覽器 Chrome

Sep 3rd, 2008

目前是測試版，下載網址：http://www.google.com/chrome/

大神出品，是不是就掛保證我是不得而知，但是速度還算不錯就是。有部分的東西不支援，像是所見即所得編輯器，還有一些 IE 的特異功能我想也不可能會支援。不過，現在已經有 FF 跟 IE 在大混戰了，再多一個瀏覽器湊一咖，是好是壞老實說我還蠻不看好的就是，畢竟這樣程式設計只會越來越難而已。


重點是，它的 JS 引擎效率之高超乎想像！開網頁幾乎都是瞬殺！沒有瞬殺一定是程式寫的不好或是主機太慢(肯定的)。

抓些畫面，它其實還蠻吃記憶體的。

Chrome 自己有工作管理員。

還有 Javascript 主控台，個人覺得，還蠻好用的。

[影像] 花蓮自由行

Sep 2nd, 2008

週末，星期五晚上搭夜車，跟 Caline 還有一個工讀生弟弟一起晃到花蓮。我只能說，柴電自強真他媽的爛！太魯閣號又買不到票，電視新聞報導的壟斷是真的，就算是你提早一個多禮拜買，一樣買不到。這種心情只有無限幹(unlimited fucking fucks)可以形容了！

大概去了這些地方，應該。這是以我的 google map 的地標，可能有一些地方沒去到沒吃到，但是 95% 以上都去到了！在地的 Caline 的表弟剛好在花蓮工作，兩天的時間跑了幾個點，其實開車應該蠻累的，因為花蓮縣實在太大了啊啊啊啊！

這個地圖比例尺是，兩公分十公里，請自行拿尺來量。

此行帶了隻小小狗。

一直都不太合群，不怎麼愛看鏡頭。
